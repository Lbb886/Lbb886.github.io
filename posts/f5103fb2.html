前提条件

在开始之前，你需要准备以下内容：

  • 一个 Telegram Bot，并获取到它的 API Token。
  • 一个可以访问的 WebDAV 服务器。
  • 基本的 JavaScript 编程知识。

步骤一：创建 Telegram Bot

  1. 打开 Telegram，搜索并启动与 @BotFather 的对话。
  2. 使用命令 /newbot 创建一个新机器人，按照提示输入机器人名称和用户名。
  3. 创建完成后，@BotFather 会给你一个 API Token，记下这个 Token。
